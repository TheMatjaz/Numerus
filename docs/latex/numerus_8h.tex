Allocates all roman numerals and their values in memory for fast conversions from value to roman numeral. 

It creates an array of pointers to strings, char$\ast$. Each char$\ast$ points to the roman numeral with the same value as the index of the char$\ast$ in the array. The returned pointer to the array points to the value 0.

Example structure\+:

\begin{DoxyVerb}*  i  | p --> numeral
* ----|---------------------
* -1  | * --> "-I"
*  0  | * --> "NULLA"
*  1  | * --> "I"
* \end{DoxyVerb}


Example usage\+:

\textcolor{keywordtype}{char} **all\_romans = \hyperlink{numerus_8c_a1985927a2efc30174ad115621a140fc9}{numerus\_allocate\_all\_romans}(1);
\textcolor{keywordtype}{char} *fortytwo = all\_romans[42];
\end{DoxyCode}
