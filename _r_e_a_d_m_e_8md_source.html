3 Numerus is a comprehensive C library that allows conversion and manipulation of
4 roman numerals - now with it's own shell.
5 
6 - [GitHub repository](https://github.com/TheMatjaz/Numerus/)
7 - [Web documentation](http://thematjaz.github.io/Numerus/)

9 
10 What is this?
11 ----------------------------------------
12 
13 Roman numerals are the numeric system used in ancient Rome based on
14 combinations of letters from the Latin alphabet to signify values.
15 [Wikipedia](https://en.wikipedia.org/wiki/Roman_numerals) has a nice article on
16 them.
17 
18 This library allows you to **convert roman numerals to values and values to
19 roman numerals**, adds some extensions to them to **support big and float
20 values** that other converters of the same kind usually don't, focuses on
21 **optimization, clean code and error handling**. Also offers a **Numerus
22 shell** to convert values on-the-fly or by pipelining (on Unix-like OS) without
23 the need to incorporate the library in an existing project.
24 
25 For a shorter explanation on how roman numerals works in Numerus and what
26 Numerus supports, check the
27 [Numerus syntax page](http://thematjaz.github.io/Numerus/md_SYNTAX.html).

30 Download, compile and use
31 ----------------------------------------
32 
33 ### 1. Download
34 
35 [Download](https://github.com/TheMatjaz/Numerus/releases) a zip/tarball of
36 the library and unpack it or clone the repository. On a Unix-like OS, you can
37 also use the command line with:
38 
39 ```sh
40 # Wget users
41 wget -qO - https://github.com/TheMatjaz/Numerus/archive/v2.0.0.tar.gz | tar -zx ; cd Numerus-*
42 
43 # Curl users
44 curl -fsSL https://github.com/TheMatjaz/Numerus/archive/v2.0.0.tar.gz | tar -zx ; cd Numerus-*
45 
46 # Git users
47 git clone https://github.com/TheMatjaz/Numerus ; cd Numerus
48 ```
49 
50 
51 ### 2a. Using the Numerus CLI
52 
53 Then, if you desire to just use Numerus directly with its build-in command line
54 interface (CLI) without incorporating it into an existing project, compile it
55 with [`cmake`](http://cmake.org/) and run it. On a Unix-like OS, do it with the
56 following one-liner in the Numerus root folder:
57 
58 ```sh
59 # Creates a `bin` directory, builds Numerus into it and runs it
60 mkdir -p bin ; cd bin ; cmake .. ; make ; ./numerus ; cd -
61 ```
62 
63 - _Hint 1:_ type the `help` command in the CLI for assistance
64 - _Hint 2:_ check the
65 [Numerus syntax page](http://thematjaz.github.io/Numerus/md_SYNTAX.html) for
66 some rules on how to write roman numerals.
67 
68 
69 ### 2b. Using Numerus in your project
70 
71 To use it as a library, add it to your project, include the
72 [Numerus header file](http://thematjaz.github.io/Numerus/numerus_8h.html) and
73 you're good to go!
74 
75 ```C
76 #include "numerus.h"
77 ```
78 
79 You'll probably need the **conversion functions**: those are the first 6
80 functions in the
81 [Numerus header file](http://thematjaz.github.io/Numerus/numerus_8h.html).
82 Check the
83 [Numerus usage examples](http://thematjaz.github.io/Numerus/md_USAGE_EXAMPLES.html)
84 for a quick tutorial on how to use Numerus and the
85 [Numerus syntax page](http://thematjaz.github.io/Numerus/md_SYNTAX.html) for
86 rules on how to write roman numerals and why Numerus uses some strange names
87 such as _long numeral_. Everything else, including functions, parameters,
88 constant and
89 [**error codes**](http://thematjaz.github.io/Numerus/numerus__error__codes_8h.html)
90 is explained in the
91 [web documentation](http://thematjaz.github.io/Numerus/)
92 (thanks [Doxygen](http://www.doxygen.org)!).

95 What's the point of this library?
96 ----------------------------------------
97 
98 Nobody is probably going to need YET another roman numerals conversion tool.
99 Numerus was created:
100 
101 - mostly for fun,
102 - as a coding exercise,
103 - to learn how to create clean code,
104 - to learn C
105 - to be able to optimize code,
106 - to understand what a good in-code and web documentation is and how to write
107  it with Doxygen,
108 - to get better at using _git_ and _GitHub_.
