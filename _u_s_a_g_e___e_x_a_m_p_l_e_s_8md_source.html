1 # Usage example #
2 
3 First of all, include the library header file:
4 
5 ```C
6 #include "numerus.h"
7 ```
8 
9 Convert a double to roman numeral with error checking.
10 -------------------------------------------------------------------------------
11 
12 Create a value and an error code variable. Pass both to the conversion function
13 `numerus_double_to_roman()` (pass a pointer to the error code). If the error
14 code is NUMERUS_OK, the conversion was successful, otherwise something bad
15 happened, like your value was outside the allowed range or `malloc()` failed.
16 
17 `numerus_explain_error()` can help to understand the error, since it prints
18 an explanation of the error for humans.
19 
20 Remember to `free()` the generated roman numeral one you don't need it anymore.
21 
22 ```C
23 double value = 12.25;
24 int errcode;
25 char *roman = numerus_double_to_roman(value, &errcode);
26 if (errcode != NUMERUS_OK) {
27  printf("Conversion failed: %s\n", numerus_explain_error(errcode));
28 } else {
29  printf("%f -> %s\n", value, roman);
30  free(roman); /* Always free() unnecessary allocated strings!!! */
31 }
32 ```

35 Convert an integer value to roman numeral without error checking
36 -------------------------------------------------------------------------------
37 **Please** be a good programmer and **always check your errors!**. If you are
38 lazy and your program explodes, don't blame me.
39 
40 Anyways, error codes can be `NULL` to ignore any error. Just pass the value to a
41 conversion function, `numerus_int_to_roman()` in this case, and get the result.
42 In case of error, the generated roman numeral will be `NULL`. If you dislike
43 error codes, you may always check this.
44 
45 ```C
46 char *roman = numerus_int_to_roman(-100, NULL);
47  /* At this point roman is NULL if anything failed in the conversion */
48 printf("%s\n", roman);
49 free(roman); /* Freeing a null pointer breaks a lot of stuff! */
50 ```

53 Convert roman numeral to an integer with some twelfths with error checking and pretty printing
54 -------------------------------------------------------------------------------
55 
56 In case of error, the generated int_part will be bigger than the maximum
57 
58 Create a roman numeral, a variable to hold a number of twelfths and an error
59 code variable. Pass all of them to the conversion function
60 `numerus_roman_to_int_part_and_twelfths()`.
61 
62 If the error code is NUMERUS_OK, the conversion was successful, otherwise
63 something bad happened, like your roman numeral had incorrect syntax (an illegal
64 character, illegal character order, missing underscore, ...). In that case the
65 returned long integer_part will be bigger than the maximum value a roman numeral
66 may have, NUMERUS_MAX_VALUE. If you dislike error codes, you may always check
67 this. `numerus_explain_error()` can help to understand the error, since it
68 prints an explanation of the error for humans.
69 
70 Remember to `free()` the generated roman numeral one you don't need it anymore.
71 
72 ```C
73 int errcode;
74 short twelfths;
75 char *roman = "-_MMVI_XCIIS...";
76 long integer_part = numerus_roman_to_int_part_and_twelfths(roman, &twelfths,
77  &errcode);
78 if (errcode != NUMERUS_OK) {
79  printf("Conversion failed: %s\n", numerus_explain_error(errcode));
80 } else {
81  char *pretty_value = numerus_create_pretty_value_as_parts(integer_part,
82  twelfths);
83  if (pretty_value == NULL) {
84  printf("%s\n", numerus_explain_error(NUMERUS_ERROR_MALLOC_FAIL));
85  } else {
86  printf("%s -> %s\n", roman, pretty_value);
87  free(roman); /* Always free() unnecessary allocated strings!!! */
88  free(pretty_value);
89  }
90 }
91 ```
